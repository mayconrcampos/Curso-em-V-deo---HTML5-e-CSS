No parãmetro type, podemos escolher o tipo de numeração da lista.

Por padrão, o type é numérico.

1 = Valor padrão

A = Cria lista em ordem alfabética maiúscula

a = Cria lista em ordem alfabética minúscula

I = Cria lista ordenada com algarismos romanos maiusculos

i = Cria lista ordenada com algarismos romanos minúsculos
