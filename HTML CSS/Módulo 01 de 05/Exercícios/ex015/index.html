Vamos aprender como utilizar Estilos CSS externos.

Estilos CSS externos são aqueles onde o código CSS não se encontra dentro do documento HTML em si e nem dentro de nenhuma de suas tags. Ou seja, todo o código referente ao estilo encontra-se em arquivo separado sob a extensão *.css
