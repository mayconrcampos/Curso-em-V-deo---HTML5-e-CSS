Texto marcado

Podemos criar um texto marcado usando a tag MARK.

para mudar a cor do fundo da marcação somente com CSS

Percebemos que se eu marcar outro trecho, a cor não será herdada do texto acima que foi marcado... Então cada marcação possui sua própria configuração de cor que, por padrão, é amarelo o fundo.

Se quiser configurar todos com uma cor fora do padrão, utilizamos um style CSS usando o seletor mark.
