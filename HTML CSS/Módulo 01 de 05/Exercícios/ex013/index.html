Vamos ver CSS Inline

Inline significa que os estilos estão sendo configurados diretamente dentro do conteúdo do documento HTML, dentro das tags.

No entanto não é utilizada, salvo para pequenos ajustes pontuais.

Os estilos aplicados neste documento foi utilizado CSS dentro de head. Também chamado CSS local.

Na próxima aula será abordado sobre CSS local.
