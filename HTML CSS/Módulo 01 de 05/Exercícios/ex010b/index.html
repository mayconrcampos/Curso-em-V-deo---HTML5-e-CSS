Criando Links para Downloads

Faça aqui o Download do PDF da Aula 10 - Ligações em toda a parte.

Usando o parâmetro type="application/pdf", o arquivo pdf não será baixado em download, pois este arquivo é reconhecido pelo próprio browser e abrirá nele.

Porém se for outro arquivo qualquer, como zip, doc, etc... O arquivo será baixado como um download comum.

Aqui vão alguns media types bem usados no nosso dia-a-dia.

  • application/zip
  • text/html
  • text/css
  • text/javascript
  • video/mp4
  • video/H264
  • video/JPEG
  • audio/aac
  • audio/mpeg
  • font/ttf
  • image/jpeg
  • image/png
